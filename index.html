Background information and approach:

  • Delaunay Triangulations

    • Delaunay triangulations maximize the minimum angle of all the angles of the triangles in the triangulation, and tend to avoid sliver triangles.

      →  


  • Incremental Method

    • The algorithm is randomized incremental, so it adds the points in random order and it maintains a Delaunay triangulation of the current point set. In order to maintain legal Delaunay triangulation, it will perform the edge flipping operation when triangulation fails to satisfy the empty circle property.

      →  


Pseudo-code:

DelaunayTriangulation(P)

Input: A set P of n points in the plane.
Output: A Delaunay triangulation of P.
  1. Create a helper triangle contains all points.
  2. for every point, p, in P do
  3.    Find the triangle abc contains p
  4.    Push edges (p, a), (p, b), (p, c) into triangluations.
  5.    SwapTest(p, a, b)
  6.    SwapTest(p, b, c)
  7.    SwapTest(p, c, a)
  8. end for
  9. Delete all edges connected to helper points from D(P).
  10. return D(P)

SwapTest(p, a, b)

  1. if (ab is an edge of helper triangle) return;
  2. find the opposite vertex d, share same edge ab with p
  3. if (d in the circle formed by p, a, b) {
  4.    Delete edge (a, b), and push edge(p, d) into D(P)
  5.    SwapTest(p, a, d)
  6.    SwapTest(p, d, b)
  7. }
